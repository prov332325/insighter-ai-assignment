# AI 기반 아동 발달 검사 결과 이해 지원 솔루션 (PoC)

인사이터(아맘때) AI 개발자 사전테스트 제출용 PoC.

K-CBCL 검사 결과 보고서를 받은 보호자가 상담사 전화 상담 전 대기 구간에서, 보고서 이해와 정서적 안정을 지원하는 Q&A 챗봇의 최소 동작 구현체.

## 핵심 동작

- 보호자가 K-CBCL 보고서에 대해 질문 → 챗봇이 5가지 설계 원칙(병리화 금지, 기준선 동반, 진단 이양, 평소대로, 상담 준비)에 따라 응답
- 보호자별 세션 ID로 대화 히스토리 저장 → 재접속 시 맥락 복원
- 입력 단 비식별화 마스킹 + 출력 단 키워드 안전 장치 작동

## 실행 방법

### 1. 사전 준비

- Python 3.10 이상
- Google AI Studio에서 API 키 발급: https://aistudio.google.com → Get API key


### 2. 프로젝트 셋업


# 레포지토리 클론
git clone https://github.com/<사용자명>/insighter-ai-assignment.git
cd insighter-ai-assignment

# 가상환경 생성 및 활성화
python -m venv venv

# Windows
venv\Scripts\activate

# Mac / Linux
source venv/bin/activate

# 패키지 설치
pip install -r requirements.txt

### 3. 환경 변수 설정

`.env.example`을 복사해 `.env` 파일 생성 후 API 키 입력.

GEMINI_API_KEY=발급받은_API_키

### 4. 실행

python chatbot.py

세션 ID 입력 후 자유롭게 대화. 종료는 `quit` 또는 `Ctrl+C`.

## 사용한 모델 / API

- **모델**: Google Gemini 2.5 Flash (`gemini-2.5-flash`)
- **API**: Google Generative AI (`google-generativeai` Python SDK)
- **선택 근거**:
  - 무료 티어 제공 (과제 FAQ의 "무료/저가 티어 사용 권장" 지침 부합)
  - 안정 GA 버전, 한국어 응답 품질 안정적
  - 빠른 응답 속도 → 챗봇 다회 호출 환경에 적합
  - 자세한 모델 선택 근거는 첨부된 기획안 4장 참조

## 예상 비용

본 PoC는 Gemini 2.5 Flash **무료 티어** 한도 내에서 운영 가능하다.

- 무료 티어 한도: 분당 15회 요청, 일 1,500회 요청
- PoC 테스트(50회 대화 가정) 시 모든 호출이 무료 한도 내

**프로덕션 확장 기준** (월 100명 보호자, 인당 평균 5턴 대화):
- 요약 생성: 100회 × 약 3,800 토큰 (캐싱으로 1회만)
- 챗봇 응답: 500턴 × 약 2,400 토큰
- 합계 약 1.58M 토큰 / 월
- Gemini 2.5 Flash 유료 단가 환산 시 **월 수 달러 수준**

## 예시 대화

샘플 보고서: 김OO 아동(만 7세 3개월), 주의집중 T=66, 내재화 T=61 (모두 준임상 범위)

### 예시 1. 결과 해석 질문 — 5원칙 작동 확인

👤 보호자: 우리 아이가 학교에서 적응을 잘 못해요. 검사지에 준임상이라고 나왔는데 문제가 있는 건가요?

🤖 챗봇: 걱정되시는 마음이 충분히 이해됩니다. 자녀분의 내재화 점수 T=61은 종합척도 기준 준임상 범위(60-62T)에 해당하며, 임상 기준(63T)에는 미치지 않아요. 지금은 평소처럼 지내시면서 상담사님과 함께 살펴볼 출발점으로 생각하시면 좋겠습니다.

작동 원칙: ① 병리화 금지 / ② 기준선 동반 / ③ 진단 이양 / ④ 평소대로 / ⑤ 상담 준비

### 예시 2. 진단 시도 — 안전 장치 작동

👤 보호자: 우리 아이가 ADHD인가요?

🤖 챗봇: 이 부분은 임상적 판단이 필요한 영역이라, 상담사님과 직접 이야기 나누시는 것이 좋겠어요.

[안전 장치 작동: 'ADHD' 키워드 감지 → 안전 응답으로 교체]

## 구현된 안전 장치

- **입력 단 비식별화**: 보고서 텍스트의 아동 실명을 `[CHILD_NAME]` 토큰으로 치환 후 LLM 호출
- **출력 단 키워드 필터**: 정신과 진단명(ADHD, 우울장애, 자폐 등) 및 치료 권고 키워드 감지 시 안전 응답으로 교체
- **마크다운 후처리**: 응답에 포함된 강조 표시(`**`) 자동 제거
- **시스템 프롬프트 절대 원칙**: 진단성 표현 생성 금지, T점수 단독 노출 금지, 호칭 가정 금지

## 한계 및 추후 개선

본 PoC는 의도적으로 범위를 좁혀 구현했다. 기획안에 제시한 다음 항목은 본 PoC에서 구현하지 않았다.

- 기능 ① 보호자용 재구성 요약 자동 생성
- Zep 기반 temporal knowledge graph 메모리 (현재는 JSON 파일로 대체)
- 응답 구조(팩트+맥락+행동 지침) 정규식 검증
- 프롬프트 인젝션 방어 정교화
- 보고서 데이터의 JSON 구조화 주입

자세한 범위 설정 근거는 기획안 5장 참조.
